**Financial Forecasting: Stock Price Prediction
Transformer-Based Deep Learning Model for Daily Stock Return Prediction:
A Comparative Study with Recurrent Neural Networks**

## Project Overview

This MSc dissertation project investigates the use of Transformer-based deep learning models for stock price prediction using historical financial time-series data.

The project aims to explore whether Transformer architectures can improve forecasting performance compared with traditional baseline models.

## Objectives

* Collect and preprocess historical stock market data
* Perform exploratory data analysis and feature engineering
* Build and train Transformer-based forecasting models
* Evaluate model performance against baseline approaches
